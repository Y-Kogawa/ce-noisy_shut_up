# TwitterとFacebookのいまどうしてる？今どんな気持ち？を消すChrome Extension

荒んでるときに見るとアスキーアートの「ねぇ、どんな気持ち？」みたいに煽られているような気がする＆いまどうしてるのかどんな気持ちなのか振り返させられているのが嫌になったのでそんな思いをぶつけてみました。


Chromeの拡張機能ページから「パッケージ化されていない拡張機能を読み込む」で読み込んでください。  
パッケージ化してもいいですね。
